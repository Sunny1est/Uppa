## 🚀 Transforme seu Foco em XP

**Uppa** é muito mais que um temporizador Pomodoro digital ou uma simples lista de tarefas. É um companheiro de produtividade projetado para ajudar jovens adultos e estudantes a atingir suas metas pessoais.

Enquanto você foca na vida real, a Uppa (uma lontra mágica na sua tela) se alimenta desse foco, evoluindo, distribuindo pontos de RPG (`INT`, `FOR`, `SAB`) e desbloqueando itens cosméticos.

### ✨ Por que usar o Uppa?
A maioria dos *Task Managers* não recompensam seu esforço bruto, apenas a finalização. Uppa recompensa **o processo**.
- 🍅 **Pomodoro Gamificado:** Cada 25 minutos focado gera Mana e XP para alimentar a Lontra.
- ⚔️ **RPG Stats:** Deseja focar em "Ler um Livro"? Ganhe pontos no seu atributo de **SAB** (Sabedoria). O sistema mapeia sua vida na forma de um gráfico de Radar (DEX, INT, FOR, CAR, CON, SAB).
- 🔄 **Tarefas Recorrentes Automatizadas:** Defina que tem "Academia toda Seg-Sex" e esqueça; o Uppa as renova automaticamente nas manhãs corretas.
- 🛒 **Economia & Loja:** Compre Poções de Foco, Poções de XP em Dobro, ou Estenda seu descanso usando seu esforço como moeda de troca.

## 🛠️ Para Desenvolvedores (Quick Start)

Se você é desenvolvedor e deseja rodar a partir do código fonte (ou contribuir com a gente):

### Pré-requisitos
- **Python 3.10+** (Recomendado 3.11 ou 3.12).
- Sistema Operacional: **Windows** (MacOS/Linux funcionam com limitações no sistema de áudio local).

### Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/Uppa.git
cd Uppa

# 2. Crie e ative o ambiente virtual
python -m venv venv
venv\Scripts\activate

# 3. Instale as dependências
pip install -r requirements.txt

# 4. Rode a aplicação
python src/main.py
```
